# bugtrckr

This will be a simple bug tracking system. When it's done ...

Features:

* Ticketsystem
* Milestones
* Projects
* Timeline
* Users

## Requirements
* PHP 5.3
* PHP Intl (for localization, recommended)
* SQLite or MySQL
