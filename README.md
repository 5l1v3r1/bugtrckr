This version is mostly full functional and we even used it as our productive bug tracking system.

# bugtrckr

This will be a simple bug tracking system. When it's done ...

Features:

* Ticketsystem
* Milestones
* Projects
* Timeline
* Users
* Localization

## Requirements
* PHP 5.3
* PHP Intl (for localization, recommended)
* SQLite or MySQL
